Steps to Run the Program:
(You will need an Eclipse IDE to load the project)

1. Clone the get repository from the URL https://github.com/Manas-Chamola/squadstack_home_activity
2. Import this project into Eclipse using File->Import->General->Existing projects into Workspace
3. Click Next
4. In Select Root Directory, browse and select the project folder wherever it is.
5. Under Projects, tick the Project name and under Working Sets, tick in Add to Working Sets and then Finish.
6. Then open the project in eclipse and open CarParkMain.java, Right click, select Run As-> Java Application and you can see the output in the console.

Note- The input file is kept inside resources folder named 'input.txt'. That file is read for input, so make changes over there to test.
